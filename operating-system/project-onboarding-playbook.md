# Playbook - Project Onboarding

## Goal

Connect a real project to the studio in a way that becomes immediately operable.

## Flow

1. create the project adapter package in `projects/<project-id>/`
2. create the local `.onivo-project/` bridge inside the project repo
3. add the project note to the vault
4. write the first strategy review
5. open the initial decision, missions, risks, and release track
6. verify that dashboards surface the new project without extra custom work
