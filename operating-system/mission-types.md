# Mission Types

## Supported Mission Classes

- `Research Mission`
- `Exploration Mission`
- `Spec Mission`
- `Plan Mission`
- `Build Mission`
- `Verification Mission`
- `Visual Mission`
- `Release Mission`
- `Recovery Mission`

## Required Fields

Every mission must define:

- owner cell
- explicit scope
- target project
- allowed action surface
- success condition
- verification path
- escalation rule
