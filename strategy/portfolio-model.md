# Portfolio Model

## Portfolio View

The studio manages a portfolio of Minecraft mod projects rather than one monolithic codebase.

Each project should eventually expose:

- maturity phase
- current focus
- health summary
- release readiness
- risk profile
- innovation potential

## Standard Project Phases

- `Incubation`
- `Discovery`
- `Pre-Production`
- `Production`
- `Hardening`
- `Release Active`
- `Maintenance`
- `Dormant`
- `Retired`
